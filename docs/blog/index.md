# 贡献提醒

欢迎大家参与到 GlobeMC 团队的博客文章的编写中来，博客文章的排版规范以本文档排版规范为准。

Frontmatter 格式如下：

```markdown
---
sidebar: false
title: 文章标题
date: 2024-06-22 19:11:00
---
```

出现在第一个 `---` （不包括 frontmatter）之前的部分为文章摘要，将显示在文章列表卡片中。
